### Added

**`argquality_dimension_profile` ‚Äî a theory-based argument-quality dimension PROFILE (ArgScope, new `argquality_dimension_profile` surface).** Segments an argument-shaped nonfiction passage and asks a pluggable LLM judge (`argquality_judge`: mock / manifest / anthropic / openai / gemini) to place, per the three top-tier Wachsmuth / GAQCorpus dimensions ‚Äî **logic** (cogency), **rhetoric** (effectiveness), **dialectic** (reasonableness) ‚Äî a COARSE descriptive `band` (`lower` / `mid` / `higher` / `null`) against the GAQCorpus rating distribution, with paragraph-anchored verbatim `evidence_spans` and a per-dimension `basis` rationale. **Descriptive PROFILE, hard no-verdict:** the deliverable is three INDEPENDENT per-dimension bands a human reads ‚Äî there is **no aggregate "argument quality" score, no `overall` band, no `score` / `aggregate` / `mean_band` / `verdict` key, no cross-dimension roll-up, and no numeric leaf anywhere under the `dimensions` block** (band/spans/basis are all strings; `distribution_reference` is a string descriptor). The bands are computed independently and NEVER summed ‚Äî no module function collapses the profile to one value. A `band` is a *distributional placement* against the corpus, **not a grade**; a `lower` band is frequently appropriate in context (a one-sided register, a rebuttal, a polemic). `null` is a first-class band (the judge declined the dimension) and is **never coerced to `lower`** (absence ‚âÝ evidence). No band is an AI-vs-human tell ‚Äî the surface refuses provenance and quality explicitly. Ships **`uncalibrated`** unconditionally ‚Äî the GAQCorpus distribution is a register-bound directional reference (research / legal / policy targets are `distant`), never a shipped threshold; the LLM judge is a prior, not a calibrated quality model. The argument-QUALITY-DIMENSION sibling of `argument_decision_audit` (structural arc) and `argument_pattern_scan` (fallacy / warrant moves); the `dialectic` band is the GLOBAL complement to `warrant_probe`'s per-claim rebuttal probe. `--judge` is **required** (no default): the `mock` backend FABRICATES bands and is for tests/CI only, so a bare run can never silently emit fake bands. `argquality_judge` defines its OWN system preamble / prompt / `fingerprint_prompt` / `_mock_judge` (not `fallacy_judge`'s or `argument_judge`'s), and its prompt fingerprint is asserted to differ from both ‚Äî so any drift gate keyed to this surface stays honest. Each `evidence_span` is verified to be a verbatim (whitespace-normalized) substring of some paragraph ‚Äî a hallucinated span the judge did not quote is dropped. Soft `register_warnings` (length floor + inferential-connective heuristic) rather than a hard register gate; fail-loud abstentions for too-short input, judge-prompt-fingerprint drift (`--expect-fingerprint`), and a missing judge SDK (`missing_dependency`). M1 is the surface + posture guards + CI contract exercised entirely through the deterministic `mock` judge (torch-free, no API, no GPU) ‚Äî there is no model-free stdlib band computation; the real-judge profile is M2 (model-gated, never GPU-gated). Rooted in: Lauscher, Ng, Napoles & Tetreault 2020, "Rhetoric, Logic, and Dialectic: Advancing Theory-based Argument Quality Assessment in Natural Language Processing" ([arXiv:2006.00843](https://arxiv.org/abs/2006.00843)). Implements `specs/30-gaqcorpus-argquality.md` (M1).

### Added

**`style_vectorizer` — an interpretable (glass-box) document vectorizer (spec 30; `voice_coherence` surface).** gram2vec ([arXiv:2406.12131](https://arxiv.org/abs/2406.12131)) is a document vectorizer in which **every dimension is a human-named stylometric feature** — function words, character n-grams, punctuation, paragraph/dialogue shape, pronoun/modal/negation rates. `style_vectorizer` emits that named-feature coordinate (`vector` + `vector_flat`) in a deterministic family-then-name order, so two documents land in the same coordinate space and a human can read the difference dimension by dimension. The feature math is **reused verbatim** from `stylometry_core.extract_features(include_spacy=False)` — no new feature code; M1 is the six stdlib families (`include_spacy=False` forced, so the default invocation is model-free and CI-runnable even when spaCy is installed). In single mode the axis list is the **full** family inventory of the target (all 135 function words, no `select_feature_names` cap) so the glass-box promise holds; with `--baseline-dir` / `--manifest` it adds a per-dimension reference distribution (mean/sd/n) and a **PROVISIONAL** descriptive band (mean ± k·sd), held-out disjoint (it warns if the target is a baseline member).

Posture: **a vectorizer, not a classifier** (`status: heuristic`). It emits **no aggregate scalar at all** — no overall distance, no score — so there is structurally nothing to threshold or rank on (the strongest no-verdict guarantee; the `results` payload carries no `is_ai`/`is_human`/`same_author`/`verdict`/`label`/`selected` key at any depth). The per-dimension band is illustrative (`calibration_status: "provisional"`), not a decision; `sd==0` yields `z=None` (never `nan`), so the R4 bounds gate passes. The claim-license refuses the authorship/AI verdict, the same-author claim, the writing-quality judgment, and use as a classifier/selection/training target. M2 (follow-up) will add the two spaCy-gated families (`pos_trigrams` / `dependency_ngrams`) behind a `--with-spacy` flag — additive named dimensions only, no new verdict. 13 tests (shape, determinism, glass-box bijection, single-mode full inventory, no-verdict recursive walk, never-selects, R4 bounds + `sd==0` edge, held-out disjoint, stdlib-import guard, drop-in registration).

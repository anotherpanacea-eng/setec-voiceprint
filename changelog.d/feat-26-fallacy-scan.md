### Added

**`fallacy_scan` — candidate rhetorical-move flags for an argument (ArgScope, new `argument_pattern_scan` surface).** Segments an argument-shaped nonfiction passage and asks a pluggable LLM judge (`fallacy_judge`: mock / manifest / anthropic / openai / gemini) to FLAG candidate rhetorical moves against the Logic 13-type fallacy taxonomy. Per flag it reports a `candidate_type`, a verbatim `span_text` pointer, and a Flee-the-Flaw implicit-logic `reconstruction` explaining *why the judge flagged it* — plus a `candidate_pattern_tally` rollup. **Descriptive, no verdict:** the data shape itself carries the "candidate"/"flag" framing (there is no `fallacy_*`, soundness, score, or aggregate key), the claim license refuses any "fallacious / unsound / weak / bad" determination, and a flagged move is frequently legitimate in context. Ships `uncalibrated` — the judge is a prior for human review, never a standalone fallacy detector. The argument-pattern sibling of `argument_decision_audit` (which scores structural arc); this flags *moves*. `--judge` is **required** (no default): the `mock` backend FABRICATES findings and is for tests/CI only, so a bare run can never silently emit fake flags. The judge's `span_text` is verified to be a verbatim (whitespace-normalized) substring of the paragraph it is attributed to — a hallucinated span the judge did not actually quote is dropped. Soft `register_warnings` (length floor + inferential-connective heuristic) rather than a hard register gate; fail-loud abstentions for too-short input, judge-prompt-fingerprint drift (`--expect-fingerprint`), and a missing judge SDK (`missing_dependency`). `fallacy_judge` defines its OWN system preamble / prompt / fingerprint (not `argument_judge`'s), so any drift gate keyed to this surface stays honest. Rooted in: Jin et al. 2022, "Logical Fallacy Detection" ([arXiv:2202.13758](https://arxiv.org/abs/2202.13758)); Hong, Sung et al. 2024, "Flee the Flaw" ([arXiv:2406.12402](https://arxiv.org/abs/2406.12402)); and (for the forthcoming M2 warrant probe) Favero et al. 2024, "Critical Questions of Thought" ([arXiv:2412.15177](https://arxiv.org/abs/2412.15177)). Implements `specs/26-fallacy-warrant-scan.md` (M1).

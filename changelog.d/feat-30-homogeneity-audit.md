### Added

**`homogeneity_audit` — pool-level set homogeneity ("AI hivemind"), M1 stdlib lens (spec 30; existing `set_level_diversity` surface).** A new *pool-level* axis: every other SETEC surface scores ONE document against a baseline, whereas `homogeneity_audit` scores a **pool of N responses** to one prompt — the signal lives *between* the texts, not inside any one. Reports the distribution of pairwise cosine similarities across the pool (the 7-key `mean`/`sd`/`min`/`p10`/`p50`/`p90`/`n` block, clean-room of `voice_fingerprint.cosine_distribution`) plus an **effective number of modes** (participation ratio over the Gram eigenvalues), with `mean_pairwise_cosine` as the headline, oriented gt = *more homogeneous*. An optional single-doc **hivemind-proximity** mode (`--target` + an operator-supplied `--centroid`/`--centroid-dir`) reports the cosine of one target to that centroid — there is **no bundled default centroid** (a shipped one would smuggle an implied verdict). M1 lens = **local-stylometric**, model-free: per-text vectors reuse `stylometry_core.function_word_features` + `char_ngram_features` (regex/`Counter`, spaCy-free) over a pool-fixed shared vocabulary. Clean-room reimplementation of the average-pairwise-cosine + mode-collapse metric from **Artificial Hivemind** (Jiang, Choi, Sap et al., NeurIPS 2025 D&B Oral; [arXiv:2510.22954](https://arxiv.org/abs/2510.22954)).

Posture: **no verdict, no band.** High homogeneity is NOT "AI" — a tight topical prompt, a shared genre/register, or a single source forces a tight pool with no AI involvement; low homogeneity is NOT "human." Like the sibling `originality_audit`, M1 emits the raw `mean_pairwise_cosine` + `effective_modes` and pushes all thresholding to the operator: there is **no absolute band**, and the paper's ~0.8 AI-regime line is recorded in `assumptions.reference_threshold_source` as the *upstream semantic-lens* figure, explicitly **not** calibrated for this stylometric lens (never a code cut). The claim-license refuses any AI/human determination, refuses reading proximity as an identity claim, and refuses comparing the local-stylometric number to the paper's `text-embedding-3-small` semantic number. `status: heuristic`, uncalibrated; the no-verdict guard is structural (a recursive `results` key walk forbids `is_ai`/`is_human`/`verdict`/`label`/`score`). Abstains (`bad_input`) below the set floor (`--min-set`, default 10). `effective_modes` needs numpy (transitive dep; declared `python_optional`) and degrades to `null` with a warning if absent — the distribution still ships. Ships with 20 stdlib tests. The **M2** LUAR / `text-embedding-3-small` semantic lenses (with the `general_imposters` output-privacy gate on the LUAR path) are a POC-gated `--lens` seam, not in this change ([arXiv:2510.22954](https://arxiv.org/abs/2510.22954); QUDsim [arXiv:2504.09373](https://arxiv.org/abs/2504.09373) is a later M3).

### Added

**`within_doc_segmentation` — within-document register-discontinuity locator (spec tier4b, M1; NEW `document_segmentation` surface).** Slides a deterministic sentence-anchored window over ONE text, measures the stylometric distance between adjacent windows (stdlib function-word + char-3/4/5-gram + sentence-shape features, the `stylometry_core` families), and emits the **internal span boundaries where the register shifts most** — each as a `stylistic_shift` / `register_discontinuity` boundary with character offsets, an ordinal shift band, and verbatim excerpts on each side. The distance profile is `d_i ∈ [0, 1]` = cosine distance over within-document z-scored feature vectors; peaks are detected at `median + k*MAD` (a within-document robust threshold, never an absolute calibrated cut). Band vocabulary: `none / slight_shift / moderate_shift / marked_shift` (no "author" token; mirrors `voice_verifier.VERIFIER_BANDS` whose strongest is `inconsistent`).

**The "never an authorship claim" guarantee is MECHANICAL.** The surface carries a module-level `AuthorshipClaimError(RuntimeError)` + `assert_no_authorship(results)` guard that recursively walks the results dict (mirroring the recursion shape of `output_schema.validate_results_bounds`) and raises on: any key in `FORBIDDEN_RESULT_KEYS` (exact, case-folded, 20 entries including `different_authors`, `n_authors`, `author`, `identity`); any string leaf value exactly matching a member; any KEY containing a `FORBIDDEN_SUBSTRINGS` token (KEY-ONLY substring walk so the honest `assumptions.confounds` value — "within-author register shifts" — passes); any `band` value outside `BAND_VOCAB`. `assert_no_authorship(results)` is called **immediately before** `build_output` in `compose_envelope`; `main()` catches `AuthorshipClaimError` and routes to `available:false` `reason_category:"policy_refused"`. The test suite includes a CI-blocking unconditional `_walk_keys` assertion on every success envelope, a happy-path-passes test (the envelope with the mandated confounds string must not raise), and a refused-attempt test (`{different_authors:[...]}` raises and routes to `policy_refused`). Cross-document comparison is **un-expressible by CLI construction**: the entrypoint exposes one input and no `--reference` / `--compare` / `--manifest` flag.

Rooted in the **PAN Multi-Author Writing Style Analysis** task lineage — the canonical sentence/paragraph-level style-change-detection benchmark. Citation: *"Overview of PAN 2026: Voight-Kampff Generative AI Detection, Text Watermarking, Multi-Author Writing Style Analysis, Generative Plagiarism Detection, and Reasoning Trajectory Detection"*, **arXiv:2602.09147**, https://arxiv.org/abs/2602.09147.

Posture: **descriptive / no-verdict / anti-Goodhart.** A register boundary is never an authorship boundary — copyedits, translations, quotation, genre-switches, and mode-switches all produce within-author register shifts. `calibration_status: provisional`. NOT a voicewright `CONSUMED_SURFACE` or a `SetecFitness` term; read-only diagnostic only. Registered as a new drop-in surface (`claim_license_surfaces/document_segmentation.txt`; no shared-dict edit). Ships with the four acceptance-blocking adversarial tests (synthetic two-author concatenation seam, prompt-injection probe, same-author-different-context corpus, happy-path-passes guard regression) plus M2-seam fail-loud test.

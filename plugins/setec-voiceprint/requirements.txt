# Runtime dependencies for setec-voiceprint.
#
# The framework runs in three tiers, with graceful degradation: every
# script's --json output reports which tier was available, and the
# variance audit's `Insufficient signal` band catches documents whose
# only-Tier-1 evidence cannot ground a band call. Required deps are the
# minimum to put Tier 2 and Tier 3 of variance_audit.py on the table.
#
# Tier 1 (sentence-length, MATTR, MTLD, Yule's K, Shannon entropy,
# FKGL, connective density, function-word ratio) runs on the standard
# library alone.
#
# Tier 2 (POS-bigram entropy + KL/JSD against baseline, MDD per
# sentence) requires spaCy and the en_core_web_sm model.
#
# Tier 3 (adjacent-sentence cosine cohesion) requires either
# sentence-transformers (preferred for calibrated cosines comparable to
# the literature's human/LLM ranges) or scikit-learn (TF-IDF fallback;
# sparser, lower absolute cosines, but useful as an embedding-free
# Tier 3 path when torch is too heavy).
#
# The validation spine (length-matched bootstrap, validation harness)
# adds SciPy, scikit-learn, and statsmodels.
#
# Install:
#
#   python3 -m venv .venv
#   source .venv/bin/activate
#   pip install -r requirements.txt
#   python -m spacy download en_core_web_sm
#
# spaCy models are not on PyPI and must be downloaded separately.

# Stylometric parsing: POS tags, dependency labels, sentence boundaries.
# Tier 2 of variance_audit.py and the POS-trigram / dependency-label
# n-gram families in stylometry_core.py.
spacy>=3.7,<4

# Length-matched bootstrap intervals for variance_audit.py and
# voice_distance.py (Phase 1 step 3); also pulled in by scikit-learn.
scipy>=1.11

# Tier 3 cohesion fallback (TF-IDF) when sentence-transformers is not
# installed, plus validation-harness primitives (Phase 1 step 4):
# roc_auc_score, precision_recall_curve, confusion_matrix,
# classification_report.
scikit-learn>=1.3

# Validation-harness confidence intervals for proportions (FPR/FNR/
# precision/recall): Wilson, Agresti-Coull, Clopper-Pearson, Jeffreys.
statsmodels>=0.14

# Optional: calibrated sentence embeddings for Tier 3 cohesion
# (Sentence-BERT / SimCSE-class). Heavier than scikit-learn TF-IDF but
# produces cosines comparable to the literature's human/LLM reference
# values (~0.30-0.55 for native fiction; ~0.50-0.70 for LLM outputs).
# Pulls in torch. Uncomment if cohesion calibration matters and the
# install size is acceptable.
# sentence-transformers>=2.7

# Optional: more accurate FKGL via textstat. variance_audit.py has a
# stdlib syllable approximation that ships now; textstat tightens the
# numbers for cross-document comparison. Uncomment if needed.
# textstat>=0.7

# Optional: NLTK is used in two graceful-degradation paths:
#   1. variance_audit.py uses nltk.tokenize.sent_tokenize when available
#      (regex fallback when not); minor quality-of-life difference.
#   2. idiolect_detector.py uses nltk.corpus.brown for the optional
#      --reference-corpus brown built-in. The detector's core scoring
#      path runs on user-supplied --reference-dir / --reference-manifest
#      corpora without NLTK.
# The collocation math in idiolect_detector.py is hand-rolled (Dunning's
# G² for bigrams, PMI for trigrams), not NLTK-based, so the package adds
# no runtime requirement for the default flow. Uncomment if you want
# either of the optional paths above. After install, run
# `python -m nltk.downloader brown` to fetch the Brown corpus data.
# nltk>=3.8

# AIC-8 / AIC-9 family (Aesthetic Authority Laundering, Closure Inflation).
#
# Foundation infrastructure shipped in 1.61.0; the AIC-8/9 detectors land
# in subsequent releases. Three additional requirements:
#
#   1. openpyxl: re-fetcher for the Brysbaert concreteness norms
#      (`scripts/fetch_brysbaert.py`). The norms ship in-repo at
#      `data/brysbaert_concreteness.csv`; openpyxl is only needed when
#      regenerating from the upstream XLSX. Listed as a soft requirement
#      because the in-repo CSV satisfies most operators.
#   2. spaCy vectors-bearing model: `en_core_web_md` (~50 MB) or
#      `en_core_web_lg` (~700 MB). The framework's other Tier 2/3 work
#      uses `en_core_web_sm` which has no word vectors. AIC-8 image-
#      conjunction and prestige-metaphor detectors require word vectors;
#      install `en_core_web_md` (preferred) via:
#         python -m spacy download en_core_web_md
#   3. requests: more reliable SSL handling than urllib for the Brysbaert
#      fetcher's macOS-Python-without-certifi failure mode. Already
#      listed in requirements-acquisition.txt; the fetcher falls back to
#      curl-via-subprocess if requests isn't installed.
openpyxl>=3.1

# PyYAML: register-typical baseline loader for AIC-8 / AIC-9.
# `scripts/register_typical_baselines.py` reads
# `baselines/register_typical.yaml`. Required when running
# `scripts/aesthetic_authority_audit.py` against a register or
# when the AIC-8/9 detectors fall back to register-typical
# defaults.
pyyaml>=6.0
